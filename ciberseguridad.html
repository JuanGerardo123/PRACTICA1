Ciberseguridad

Tipos de malware

Tipo de malware Función Impacto
Virus Self replicate, infect files Daño de datos al igual que de sistemas
Gusanos Se propaga a travez de las redes Interrupcion de la red, agotamiento de recursos
Caballo de troya Se disfraza de software legal Robo de datos, control del sistema
Ransomware Se disfraza de software legitimo Robo de datos, control del sistema
Software espía Supervisa en secreto la informacion del usuario Violacion de privacidad, robo de datos
Software Publicitario Muestra anuncios no deseados Molesta, degradacion del rendimento
Rootkit Ocultar software malicioso Acceso no autorizado, amenazas persistentes
Keylogger Registra las pulsaciones de teclas Robo de credenciales, filtracion de datos
Fileless malware
malware sin archivos		 Corre de la memoria Ataque sigiloso y dificil de detectar
Cryptojacker Minas de criptomonedas problemas de rendimiento, secuestro de recursos
Red de boots (Botnet) Crear una red de dispositivos infectados Ataque DDos, distribucion de spam
Bomba Logica (logic bomb) Ejecuta codigo en el disparador Destruccion de datos, danos al sistema
Wiper (limpiabrisas) Destruye datos, daños al sistema Perdida irreversible de datos
Scareware
