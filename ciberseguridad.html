Ciberseguridad

Tipos de Malware

Tipo de Malware Función Impacto
Virus Se replica automáticamente e infecta archivos. Daño a archivos y sistemas.
Gusanos Se propagan por la red sin intervención. Consumo de ancho de banda, colapso del sistema.
Caballo de Troya Se oculta como software legítimo. Robo de información o control remoto.
Ransomware Secuestra datos y exige un rescate. Pérdida de datos, impacto económico.
Spyware Espía la actividad del usuario. Violación de privacidad.
Adware Inunda con anuncios. Molestias, ralentización del sistema.
Rootkit Oculta malware para evitar su detección. Acceso persistente no autorizado.
Keylogger Captura lo que escribe el usuario. Robo de contraseñas y datos sensibles.
Fileless Malware Opera desde la memoria RAM. Difícil de detectar, muy sigiloso.
Cryptojacker Usa el sistema para minar criptomonedas. Rendimiento degradado, alto consumo de energía.
Botnet Red de dispositivos infectados. Lanzan ataques como DDoS.
Bomba lógica Se activa con una condición específica. Elimina o daña datos.
Wiper Destruye archivos y sistemas. Pérdida total de datos.
Scareware Asusta al usuario con alertas falsas. Hace que instalen software no deseado.
